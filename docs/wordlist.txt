modositas:

kell relacioba kossem a kulombozo generacios szavakat...
kifejtem egy peldaval:
van egy szolista, peldaul: angol szavak
ehez tartonak a kulombozo generacios szavak, peldaul:
GEN1: one, two, free, six, ...
GEN2: move, sleep, work, go, run, ...
Gen3: blue, black, red, purple, green, ...

a GEN1 bol kiindilva a szvalat relacioba kell hozni a GEN2 bel levio szavakkal nem kell mindent caak mit akarunk
GEN1 - GEN2
peldaul: one - sleep, one - work, one - go
six - move, six - go, six - run
es igy tovabb
Most jon a GEN2 - GEN3
move - red, move - green, move - blue
sleep - blue, sleep - black, sleep - purple
work - blue, work, green, ...

es igy tovabb...
ahany generacio van addig lehessen folytatni
a relaciokat csak a kisseb generaciotol egyel nagyobig lehet megadni
tehet: GEN1 - GEN2, GEN2 - GEN3, GEN3 - GEN4, ...
